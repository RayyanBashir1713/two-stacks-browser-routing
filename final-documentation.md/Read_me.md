# Browser Navigation Using Dual Stacks

## Project Summary
This project simulates a browser navigation system using two stacks to manage
backward and forward navigation efficiently.

## Data Structure Used
- Stack (LIFO)

## Key Features
- Visit new pages
- Back and forward navigation
- Update current page
- Search page in history
- Display complete navigation history

## How to Run
1. Compile using a C++ compiler
2. Run the executable
3. Use menu-driven interface
